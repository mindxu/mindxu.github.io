CAT3 College Simulation Game

While browsing Twitter, I found a post announcing the release of GameBoy (GB) Studio to celebrate Gameboy's 30 Anniversary.

In GB Studio, you can create retro-style games using pseudocode. There is no coding skill or experience required, and there are tutorials on YouTube to help get set up. It also comes with a sample project with sprites and backgrounds, which were used to create this college game.
